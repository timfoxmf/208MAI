Задача Коши и формулировка теоремы Коши для ДУ первого порядка, разрешённых и не разрешённых относительно производной.

Общяя формулировка задачи Коши

В общем случае решения дифференциального уравнения первого порядка подразумевает получения семейства решения этого уравнения. А задача Коши напротив выделяет с помощью начальных условиях то решения из этого семейства которое соответствует данным условиям. Т.е по сути является частным решение ДУ уравнения.

Задача Коши. Найти решения $y = y(x)$ дифференциального уравнения $F(x,y,y') = 0$, или в частном случае — уравнения $y'=f(x,y)$, удовлетворяющему условию $y(x_0) = y_0$, где числа $x_0 , y_0$ задана по условию.

Формулировка теоремы для ДУ 1-го порядка разрешённых относительно производной.

Теорема Коши (для уравнения 1-го порядка разрешённых относительно производной). Пусть в некоторой области $G$ изменения своих переменных функция $f(x,y)$ непрерывна. Тогда для любой внутренний точки этой области $(x_0, y_0) \in G $ можно указать такой замкнутый интервал: $$x_0 - h \leq x \leq x_0 + h, h > 0$$

Оси $Ox$ где существует решения $y = y(x)$ дифференциального уравнения $y'=f(x,y)$, удовлетворяющему условию $y(x_0) = y_0$. Если кроме того в этой области непрерывна производная $ \frac{\partial f}{\partial y}$, то указанное решения является единственным.

Замечания.

  1. Условие непрерывности функции $f(x,y)$ является условие существования Задачи Коши.
  2. Условие непрерывной производной в этой области ($\frac{\partial f}{\partial y}$), является условием единственности решения Задачи Коши.

Формулировка теоремы Коши для ДУ уравнений 1-го порядка не разрешённых относительно производных.

Теорема Коши (для уравнения 1-го порядка не разрешённого относительно производных). Пусть в некоторой области $G$ изменения своих аргументов функция $F(x,y,y')$и её производная $\frac{\partial F}{\partial y'}$ непревны и кроме того $\frac{\partial F}{\partial y'} \neq 0$. Тогда для любой внутренний области точки $(x_0, y_0) \in G$ можно указать такой замкнутый интервал:

$$ x_0 - h\leq x \leq x_0+h, h>0$$

Оси $Ox$ на котором существует решения $y = y(x)$ дифференциального уравнения $F(x, y, y') = 0$ удовлетворяющему условию $y(x_0) = y_0$ если кроме того в этой области непрерывна производная: $$\frac{\partial y'}{\partial y}=-\frac{\frac{\partial F}{\partial y}}{\frac{\partial F}{\partial y'}}$$

То для любого значения $y'_0$ являющейся корнем уравнения $F(x_0, y_0, y') = 0$, такое решения удовлетворяющие условию $y(x_0) = y_0$ и $y'(x_0) = y'_0$ является единственным.
